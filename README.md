🔗 Acceso / Demo: No disponible

📋 Descripción:
Este proyecto es un ejercicio práctico diseñado para introducir al usuario en el **aprendizaje supervisado** y su aplicación en la **clasificación de imágenes**. Utilizando la plataforma accesible **Google Teachable Machine**, se guía paso a paso el proceso de entrenar un modelo que pueda distinguir entre imágenes de perros y gatos. El objetivo principal es comprender cómo una inteligencia artificial aprende a clasificar información visual a través de ejemplos etiquetados.

La metodología enfatiza la importancia de recolectar un conjunto de datos variado para lograr un entrenamiento robusto del modelo, así como la evaluación continua para identificar limitaciones y oportunidades de mejora. Es una excelente introducción práctica al Machine Learning sin necesidad de codificación avanzada, ideal para explorar conceptos fundamentales de IA.

✨ Funcionalidades:
| Funcionalidad | Descripción |
| --- | --- |
| Clasificación de Imágenes | Utiliza Google Teachable Machine para clasificar imágenes en perros y gatos |
| Entrenamiento del Modelo | Entrenar un modelo con imágenes descargadas de Internet |
| Pruebas del Modelo | Probar el modelo con imágenes nuevas para evaluar su precisión |

🚀 Uso:
El uso del proyecto se centra en seguir los pasos de la guía para interactuar con la plataforma web de Google Teachable Machine:
1.  Acceder a Google Teachable Machine (`https://teachablemachine.withgoogle.com`).
2.  Crear un nuevo proyecto de clasificación de imágenes (`Image Project`).
3.  Definir las categorías de clasificación, por ejemplo, `Perro` y `Gato`.
4.  Subir un conjunto diverso de imágenes de perros y gatos para el entrenamiento del modelo.
5.  Iniciar el proceso de entrenamiento del modelo.
6.  Probar el modelo con nuevas imágenes para evaluar su precisión y identificar sus limitaciones.
