_*Entrenamiento de un modelo de clasificación de imágenes con Google Teachable Machine*_ 

🔗 Acceso / Demo: No disponible

📋 Descripción:
Este proyecto se enfoca en el aprendizaje supervisado utilizando Google Teachable Machine para clasificar imágenes de perros y gatos. El objetivo es entrenar un modelo que pueda distinguir entre estas dos categorías.

✨ Funcionalidades:
| Funcionalidad | Descripción |
| --- | --- |
| Clasificación de Imágenes | Utiliza Google Teachable Machine para clasificar imágenes en perros y gatos |
| Entrenamiento del Modelo | Entrenar un modelo con imágenes descargadas de Internet |
| Pruebas del Modelo | Probar el modelo con imágenes nuevas para evaluar su precisión |

⚙️ Instalación:
No requiere instalación de software adicional. Acceder a [https://teachablemachine.withgoogle.com](https://teachablemachine.withgoogle.com) para comenzar.

🚀 Uso:
1. Acceder a Google Teachable Machine.
2. Crear un proyecto de clasificación de imágenes.
3. Subir imágenes de perros y gatos.
4. Entrenar el modelo.
5. Probar el modelo con imágenes nuevas.
