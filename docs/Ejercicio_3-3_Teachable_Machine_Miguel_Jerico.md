# Ejercicio 3.3: Aprendizaje Supervisado con Google Teachable Machine
## 📋 Resumen
Este ejercicio se enfoca en el aprendizaje supervisado con Google Teachable Machine, específicamente en la clasificación de imágenes de perros y gatos. El objetivo es entrenar un modelo para distinguir entre estas dos categorías utilizando la herramienta Google Teachable Machine. A través de este ejercicio, se busca comprender el concepto de aprendizaje supervisado y cómo se aplica en la clasificación de imágenes.

## 🔑 Puntos clave
- Utilización de Google Teachable Machine para el aprendizaje supervisado.
- Clasificación de imágenes en dos categorías: perros y gatos.
- Proceso de trabajo estructurado que incluye la creación de un proyecto, elección de categorías, búsqueda y descarga de imágenes, importación de imágenes, entrenamiento del modelo, prueba del modelo, búsqueda de errores y mejora del modelo.

## 📝 Detalle
### Objetivo
El objetivo de este ejercicio es entrenar un modelo de clasificación de imágenes supervisado utilizando Google Teachable Machine para distinguir entre imágenes de perros y gatos.

### Herramienta Utilizada
- **Google Teachable Machine (Standard Image Model)**: Es una plataforma de aprendizaje automático accesible y fácil de usar, ideal para estudiantes y principiantes.

### Proceso de Trabajo
1. **Acceder a Teachable Machine**: 
   - Acceder a [https://teachablemachine.withgoogle.com](https://teachablemachine.withgoogle.com).
   - Pulsar "Get Started".
   - Seleccionar "Image Project" y luego "Standard Image Model".

2. **Elegir las Categorías**:
   - Renombrar las clases como:
     - Clase 1 → **Perro**
     - Clase 2 → **Gato**

3. **Buscar Imágenes**:
   - Buscar imágenes de perros y gatos en Google Imágenes.
   - Descargar aproximadamente 20 imágenes diferentes de cada categoría, procurando variedad en razas, colores, posiciones, fondos y tamaños.

4. **Importar las Imágenes**:
   - En Teachable Machine, pulsar "Upload".
   - Seleccionar todas las imágenes de la carpeta "Perro" y luego las de la carpeta "Gato".

5. **Entrenar el Modelo**:
   - Pulsar "Train Model".
   - Esperar unos segundos mientras la IA aprende.

6. **Probar el Modelo**:
   - Pulsar "Upload Image".
   - Descargar imágenes de perros y gatos que no se utilizaron durante el entrenamiento y probarlas.

7. **Buscar Errores**:
   - Intentar "engañar" al modelo con imágenes de lobos, zorros, cachorros, peluches de perros y dibujos animados de gatos.

8. **Mejorar el Modelo**:
   - Descargar 20 imágenes adicionales de cada animal con más variedad.
   - Entrenar nuevamente el modelo.

## ✅ Conclusiones / siguientes pasos
- El ejercicio permite una comprensión profunda del aprendizaje supervisado y su aplicación en la clasificación de imágenes.
- Google Teachable Machine es una herramienta poderosa y accesible para proyectos de aprendizaje automático.
- La mejora continua del modelo es crucial para aumentar su precisión y capacidad de generalización.
